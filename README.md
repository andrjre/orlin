# orlin
An app to centralise common productivity tools.

## Vision
Nowadays productivity apps are decentralised and spread across multiple devices, interfaces and subscriptions.
Orlin intends to centralise these apps to create a one-stop-shop to help maintain routine, planning and learning.
Written using flutter for the best cross-platform capabilities, orlin is intended to be lightweight, open-source and free.

## Stack
- Flutter / Dart
- SQLite via Drift
- Riverpod
